# Portofolio Ahmad Davfa Maulana

Website portofolio statis siap hosting. Dibuat dengan HTML, CSS, dan JavaScript tanpa build tools.

## Struktur

- `index.html` — halaman utama portofolio.
- `assets/css/style.css` — styling 3D, responsive layout, animasi.
- `assets/js/main.js` — tilt interaction, lightbox gallery, scroll reveal, particle canvas.
- `assets/img/` — foto profil, logo, dan screenshot proyek.
- `assets/docs/Ahmad-Davfa-Maulana-CV.pdf` — file CV untuk tombol download.

## Cara hosting

1. Upload semua isi folder ini ke hosting statis seperti Netlify, Vercel, GitHub Pages, atau shared hosting.
2. Pastikan file `index.html` berada di root folder hosting.
3. Tidak perlu database untuk portofolio ini karena semua konten sudah statis.

## Edit cepat

- Ubah teks profil atau kontak di `index.html`.
- Ubah warna dan efek 3D di `assets/css/style.css` pada bagian `:root`.
- Tambah screenshot proyek baru di `assets/img/projects/`, lalu tambahkan item galeri di section `projects` pada `index.html`.
